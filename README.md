# Smart Budget Tracker (Java)

A console-based budget management application built using **Core Java and Object-Oriented Programming (OOP)**.

The system allows users to record daily expenses, categorize spending, analyze financial behavior, and manage a monthly budget.

---

## Features

- User login authentication
- Add expense with date, category, amount, and note
- Quick expense entry
- View all expenses
- Monthly spending summary
- Category-wise spending summary
- Most used spending category
- Highest spending category
- Highest single expense
- Financial personality analysis
- Search expenses by date or category
- Edit and delete expenses
- Remaining budget display

---

## Technologies Used

- Java (Core Java)
- Object-Oriented Programming (OOP)
- Arrays for expense storage
- Console-based user interface

---

## OOP Concepts Implemented

- **Encapsulation**
- **Inheritance**
- **Classes and Objects**
- **Constructor usage**

Example:

- `BudgetItem` → Base class  
- `Expense` → Child class  
- `SmartBudget` → Business logic class
